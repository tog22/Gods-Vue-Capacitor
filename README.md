# ⚡️ Torchbearers

## Project setup
```
npm install
npm i --save  @capacitor-firebase/messaging firebase mitt
```

## Using Capacitor
```
npm run build
npx cap sync
npx cap run android
```

#### Generating icons
```
npm run resources
```

Based on contents of resources directory.
